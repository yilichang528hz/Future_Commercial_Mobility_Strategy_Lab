# Scenario Simulator Interactive Map

Standalone vanilla HTML/CSS/JavaScript dynamic module generated from `scenario-simulator-interactive-map.md`.

## Preview
Open `index.html` directly or deploy the folder on GitHub Pages.

## Data discipline
Uses only local illustrative demo data. No package manager, no backend, no CDN, no API calls, no telemetry, no map tiles, and no live data.

## Accessibility
Includes keyboard-operable controls, visible focus states, aria-live text updates, SVG roles, and reduced-motion handling.
