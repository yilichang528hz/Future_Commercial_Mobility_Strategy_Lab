Home-only note

Creative Motivation

This lab turns future mobility signals into practical planning tools for commercial vehicles, linking Taiwan-based operating contexts, energy transition, human-centered logistics, safety, connected fleets, ESG, and responsible AI strategy.

Global operating rules

Static portfolio architecture with demo-data discipline

The site separates strategy hypotheses from verified facts, keeps generated data clearly labeled, and avoids official-brand confusion.

Trilingual UI

Every public label, control, chart caption, disclaimer, and glossary entry is driven by language keys.

Four themes

Theme switching uses stable body theme tokens without resetting content or interactive state.

Demo data first

Scores and scenarios are illustrative unless the source registry explicitly supports a claim.

Strategy surfaces

Module shells ready for deeper content sections

Each module inherits the same language, theme, glossary, image-governance, and writing-safeguard systems.
